# Wireframe Docs Index

이 디렉터리는 wireframe 관련 문서의 단일 진입점이다.

## Canonical Docs

- `docs/wireframe/spec.md`
  - 제품/UX 요구사항, 화면 구조, 글로벌 패턴
- `docs/wireframe/coverage-cutover.md`
  - frontend 대비 커버리지 체크 + cutover 게이트/DoD
- `docs/wireframe/runtime-contract.md`
  - API/WS 계약, mock layer 설계, 상태 모델

## Rule

- wireframe 관련 신규/수정 문서는 위 3개 canonical 문서 우선 갱신
- 분석성 임시 문서는 장기 보관하지 않고 필요한 내용만 canonical 문서에 흡수
